# Structural Simplification in Popular Music

This repository contains a data science project investigating whether commercially successful songs tend to be structurally simpler than non-charting songs.

The project compares Spotify songs with lyrics against Billboard Hot 100 chart records. Structural simplicity is measured through song duration, lyrical repetition, and lexical diversity. The main research question is:

**Do commercially successful songs tend to be structurally simpler than non-charting songs?**

## Project Structure

- `data_exploration.ipynb` - loads the raw datasets, explores them, cleans the data, matches Spotify songs with Billboard chart records, and creates the prepared analysis file.
- `research_report.ipynb` - final report notebook with research framing, methodology, feature engineering, visual analysis, statistical testing, results, limitations, and conclusion.
- `spotify_billboard_prepared.csv` - generated by `data_exploration.ipynb`. This file is not intended to be stored permanently if raw CSV files are removed from the repository.

## Data Sources

The CSV files are not included in the final repository. To run the project, download the datasets manually and place the required CSV files in the project root directory.

Required raw files:

- `spotify_songs.csv`
- `charts.csv`

Dataset links:

- Spotify songs with audio features and lyrics: https://www.kaggle.com/datasets/imuhammad/audio-features-and-lyrics-of-spotify-songs/code/data
- Billboard Hot 100 chart dataset: https://www.kaggle.com/datasets/dhruvildave/billboard-the-hot-100-songs


## Requirements

Python version:

- Python 3.12

Python libraries:

- pandas
- numpy
- matplotlib
- seaborn
- scipy
- jupyter

Optional, depending on your notebook environment:

- ipykernel

## Setup

Create and activate a virtual environment:

```bash
python3.12 -m venv myenv
source myenv/bin/activate
```

Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn scipy jupyter ipykernel
```

Start Jupyter:

```bash
jupyter notebook
```

## How to Run the Project

1. Download the two datasets from the links above.
2. Place the files `spotify_songs.csv` and `charts.csv` in the root folder of this repository.
3. Run `data_exploration.ipynb` first. This notebook creates `spotify_billboard_prepared.csv`.
4. After `spotify_billboard_prepared.csv` has been created, run `research_report.ipynb`.

The report notebook depends on `spotify_billboard_prepared.csv`, so it will not run correctly until the data exploration notebook has been executed.
